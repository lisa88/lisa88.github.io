css选择器—属性选择器

发表于 2015-11-02   |   分类于 CSS

属性选择器的主要作用是对带有指定属性的HTML元素设置样式。使用CSS3属性选择器，你可以只指定元素的某个属性，或者你还可以同时指定元素的某个属性和其对应的属性值

属性选择器

IE6不支持属性选择器

1，E【attr】

选择.main下面所有带class的p元素
选择.main下面同时具有class，title两个属性的p元素

2，E【attr=”value”】

E[attr=”value”]是指定了属性值“value”

选择.main下面class=text的p元素。
属性和属性值必须完全匹配，特别是对于属性值是词列表的形式时

3，E【attr~=”value”】

这种属性选择器是属性值是一个或多个词列表，如果是列表时，他们需要用空格隔开，只要属性值中有一个value相匹配就可以选中该元素。

E[attr=”value”]和E[attr~=”value”]之间的区别，有波浪（〜）时属性值有value就可以匹配，没有波浪（〜）时属性值要完全是value时才匹配。

4，E【attr^=”value”】

选择attr属性值以”value”开头的所有元素

5，E【attr$=”value”】

选择元素attr属性，并且他的属性值是以value结尾的

6，E【attr*=”value”】

选择attr属性值中包含子串”value”的所有元素。只要你所选择的属性，其属性值中有这个”value”值都将被选中。

7，E【attr|=”value”】

选择attr属性值等于value或以value-开头的所有元素

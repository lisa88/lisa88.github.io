sass基础用法二

上一节的sass基础用法一主要是从sass文件后缀名、嵌套、变量、编译等进行了介绍。这篇主要介绍mixin、继承、%点位符、函数等。
建议大家对文章中所有举的例子自己都能在sassmeister上编译一下，加深理解。

sass-mixin(混合宏)

sass中使用@mixin声明混合，可以传递参数，参数名以$符号开始，多个参数以逗号分开，也可以给参数设置默认值。声明的@mixin通过@include来调用。

一，mixin参数类型

1，无参数的mixin


@mixin title-style {
    margin: 0 0 20px 0;
    font-size: 20px;
    font-weight: bold;
    text-transform: uppercase;
}
//调用
.test{
    @include title-style;
}

2，一个参数的mixin

@mixin title-style($color) {
    margin: 0 0 20px 0;
    font-size: 20px;
    font-weight: bold;
    text-transform: uppercase;
    color: $color;
}
//调用
.test{
    @include title-style(#ccc);
}

3，多参数的mixin

@mixin title-style($color, $background) {
    margin: 0 0 20px 0;
    font-size: 20px;
    font-weight: bold;
    text-transform: uppercase;
    color: $color;
    background: $background;
}
//调用
.test{
    @include title-style(#ccc, #f50);
}

4，默认的参数

当你使用mixin的参数时，定义一个经常使用的值作为默认参数是很方便的，你只需要简单的调用mixin，不用传递参数，如果有默认的值，会把这个值传递过去。 
@mixin title-style($color, $background: #eee) {
    margin: 0 0 20px 0;
    font-size: 20px;
    font-weight: bold;
    text-transform: uppercase;
    color: $color;
    background: $background;
}
//调用
.test{
    @include title-style(#ccc);
}
//或是你可以想覆盖，传你需要的值
.test{
    @include title-style(#ccc,#000);
}
当你有多个默认参数在mixin中定义，你可以有选择的覆盖默认参数，而不需要全部重新定义。 
@mixin title-style($color: blue, $background: green) {
    margin: 0 0 20px 0;
    font-size: 20px;
    font-weight: bold;
    text-transform: uppercase;
    color: $color;
    background: $background;
}
//调用
.test{
    @include title-style($background:pink);
}

@content

@content可以用来解决css3的@media带来的问题。它可以使@mixin接受一整块样式，接受的样式从@content开始。 
@mixin header{
    .test{
        @content;
    }
}
@include header{
    width:1000px;
    height:200px;
    .logo{
        width:200px;
    }
}

二，mixin实例展示

解决浏览器前缀-单一参数

@mixin rounded($radius) {
    -webkit-border-radius: $radius;
       -moz-border-radius: $radius;
            border-radius: $radius;
}
//调用
.test{
    @include rounded(10px);
}

解决浏览器前缀-多个参数

@mixin shadow($x, $y, $blur, $color) {
    -webkit-box-shadow: $x $y $blur $color;
       -moz-box-shadow: $x $y $blur $color;
            box-shadow: $x $y $blur $color;
}
//调用
.test{
    @include shadow(0, 1px, 2px, rgba(0,0,0,.5));
}

//box-shadow设置很多个值 ，传递参数的后面加上...
@mixin box-shadow($shadow...){
    -webkit-box-shadow:$shadow;
    -moz-box-shadow:$shadow;
    box-shadow:$shadow;
}
//调用
.test{
    @include box-shadow(0 0 5px rgba(0,0,0,.3),inset 0 0 3px rgba(255,255,255,.5));
}

定义media-queries的最小最大宽度

//选择器里面调用，强调以选择器为主
@mixin screen($res-min, $res-max){
  @media screen and ( min-width: $res-min ) and ( max-width: $res-max ){
    @content;
  }
}
#header{
    @include screen(780px,1000px){
        color:red;
    }
}

//直接调用，强调以屏幕大小为主，可以方便组织所有在这个条件中的都写在一起
//如果做响应式布局，我们建议使用第二种方法，以断点为主来写样式，把某个断点下的样式全部写在一起
@include screen(780px,1000px){
    body{
        font-size:14px;
    }
}
